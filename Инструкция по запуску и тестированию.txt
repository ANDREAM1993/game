Инструкция по запуску программы:

1) Скачать все файлы программы на локальный компьютер.

2) Пометить все файлы в одну папку (3 файла *.DAT, 2 файлa *.ТХТ, 3 файла *.PY).

3) Используя коммандную строку (CMD) перейти в папку с файлами программы (в шаге 2).

4) Запустить игру в коммандной строке (CMD) коммандой: python game_main.py (открытие файла game_main.py).

5) Проверка игры:

5.1) Для проверки игры согласно задания (автоигра) выбрать 2 пункт (ТЕСТ) главного меню. Игра закончится автоматически
по достижению уровня здоровья одного из игроков 0%. История ходов будет записана в файл last_game_history.txt.

5.2) Для самостоятельной игры (против компьютера) выбрать 1 пункт в главном меню (ИГРАТЬ). Результаты игры (журнал ходов)
также будет сохранен в файл last_game_history.txt (журнал предыдущей игры будет удален).

5.3) Для вывода информации о программе выбрать 3 пункт главного меню (СПРАВКА).

5.4) Для вывода из программы выбрать 4 пункт главного меню (ВЫХОД).
